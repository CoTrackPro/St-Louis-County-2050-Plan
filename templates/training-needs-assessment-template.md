# Training Needs Assessment Template

Use this template to conduct a systematic training needs assessment (TNA)
at the county-wide, department, or role-group level. Connects organizational
goals and KPI targets to workforce capability gaps, producing a prioritized
training investment plan.

Load `references/workforce-development.md` for competency framework and
modality guidance.

## 3. Needs Prioritization Matrix

Score each identified need on three dimensions (1–5 scale):

| # | Training Need | KPI Impact (1–5) | Urgency (1–5) | Reach (# staff) | Priority Score | KSAB Domain |
|---|--------------|-------------------|--------------|-----------------|----------------|-------------|
| 1 | [Need] | [X] | [X] | [N] | [Impact × Urgency] | [K/S/A/B] |
| 2 | [Need] | [X] | [X] | [N] | [Score] | [K/S/A/B] |
| 3 | [Need] | [X] | [X] | [N] | [Score] | [K/S/A/B] |
| 4 | | | | | | |
| 5 | | | | | | |
| ... | | | | | | |

**Scoring guide**:
- **KPI Impact**: 5 = directly linked to red-status KPI; 1 = nice to have
- **Urgency**: 5 = compliance deadline or safety risk; 1 = long-term development
- **Reach**: Raw count of affected staff
- **Priority Score**: Impact × Urgency (max 25). Rank by score; break ties with Reach.

## Completion Instructions

1. Gather input from all sources in Section 2 before prioritizing.
2. Use the prioritization matrix (Section 3) to rank needs objectively.
3. Match each need to the correct KSAB domain — this determines modality.
4. Separate mandatory/compliance items (Section 4) from discretionary.
5. Build budget that's defensible for council approval.
6. Set evaluation criteria *before* delivering any training.
7. All illustrative data must be labeled `[ILLUSTRATIVE]`.

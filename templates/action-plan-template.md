# Action Plan Template — KPI Improvement Initiative

Use this template when a department needs to improve a specific KPI.
Produces a structured improvement plan with root cause analysis,
interventions, milestones, and measurement plan.

---

## Completion Instructions

1. Identify the underperforming KPI from department scorecard.
2. Conduct root cause analysis with department staff (not in isolation).
3. Brainstorm interventions and score using Impact × Feasibility matrix.
4. Select top 3–4 interventions for the 90-day plan.
5. Assign owners and milestone dates.
6. Estimate resource requirements and identify funding.
7. Present to department head for approval before implementation.

## ASSUMPTIONS (when real data unavailable)
- Label all estimated values as `[ILLUSTRATIVE]`
- Use national municipal medians from `assets/benchmark-data.csv` as proxy
- Flag data gaps that must be filled before plan execution
